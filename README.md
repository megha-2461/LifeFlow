## Run Locally

Clone the project.


Go to the project directory:

```bash
cd LifeFlow/client
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

Runs the app in the development mode.

Open http://localhost:3000 to view it in your browser.
